Список необходимых тестов для проверки работоспособности системы:
1. Тесты на некорректный пользовательский ввод:
    1) Поведение системы при вводе несуществующего IP-адреса сервера
    2) Поведение системы при вводе неверного логина и/или пароля для подключения к серверу
    3) Поведение системы при отсутствии какого-либо из элементов, необходимых для подключения

2. Тесты на проверку сохранности данных:
    1) Поведение системы при отключении сервера во время работы клиента

3. Тесты на некорректные данные в файле:
    1) Поведение системы если файл пустой или не имеет необходимых ключей
    2) Поведение системы если файла нет на сервере
    3) Поведение системы при наличии наименований ключей в значениях (НЕ РЕАЛИЗОВАНО ДЛЯ ЭКОНОМИИ ВРЕМЕНИ НА ПАРСИНГЕ (библиотеки то нельзя юзать))
